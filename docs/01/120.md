# Алгоритм - основа програми

## Визначення

!!! info ""

    Алгоритм - це послідовність дій, що призведе до бажаного результату.

## Приклади

???+ example "1"
    1. Розблокувати екран
    2. Запустити програму
    3. Набрати текст повідомлення
    4. Надіслати повідомлення другу/подрузі
    5. Дочекатися відповіді
    6. Прочитати відповідь

???+ example "2"
    1. Запустити гру
    2. Розважитись півгодини
    3. Закрити гру
    4. Відкрити зошит для домашки
    5. Закрити зошит для домашки
    6. Повернутись до пункту 1

???+ bug "3"
    1. Запустити програму
    2. Розблокувати екран
    3. Відправити повідомлення
    4. Написати текст повідомлення

---

!!! abstract "python :simple-python: example"
    ```python
    print(10)
    print(14)
    print(3)
    ```
??? quote "Виведе"
    ```
    10
    14
    3
    ```
