# :material-numeric: Цілі числа та операції над ними

## Створення цілого числа та друк типу даних

!!! abstract "python :simple-python: example"
    ```python
    i = 1 #(1)!
    print(i) #(2)!

    variable_type = type(i) #(3)!
    print(variable_type) #(4)!

    variable_type_name = variable_type.__name__ #(5)!
    print(variable_type_name) #(6)!
    ```

    1. присвоює ім'я `i` значенню `1`
    2. друк значення на яке вказує ім'я `i`
    3. присвоює ім'я `variable_type` значенню виразу праворуч від `=`
    4. друк значення на яке вказує ім'я `variable_type`
    5. присвоює ім'я `variable_type_name` значенню виразу праворуч від `=`
    6. друк значення на яке вказує ім'я `variable_type_name`

??? quote "Виведе"
    ```
    1
    <class 'int'>
    int
    ```
## Коли змінна створена

Доступні такі дії

### дати змінній псевдонім

!!! abstract "python :simple-python: example"
    ```python
    my_year_of_birth = 1990
    my_year = my_year_of_birth # (1)!

    print(my_year_of_birth)
    print(my_year)

    print(hex(id(my_year)))
    print(hex(id(my_year_of_birth))) # (2)!

    my_year = 2000 # (3)!

    print(my_year_of_birth)
    print(my_year)

    print(hex(id(my_year_of_birth))) 
    print(hex(id(my_year))) # (4)!
    ```

    1. `my_year` це лиш псевдонім `my_year_of_birth`
    2. ==Адреса== в пам'яті обох змінних ==однакова==
    3. тут створиться ==нова змінна==, `my_year` вже НЕ псевдонім
    4. в нової змінної ==відрізняється адреса== в пам'яті
