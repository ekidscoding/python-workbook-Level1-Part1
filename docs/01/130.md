# Змінні - наклейки, а НЕ коробки

Змінні - це основа основ програмування.

!!! info ""
    Змінна - це така собі наліпка, стікер :material-sticker-text: або бірка :label:, яка вішається на щось.

**Щось** -- це одне з:

- ціле число (3, 14, 159265359)
- десятковий дріб, наприклад :material-pi: (3.14159265359)
- рядок тексту ("Python", "Друг", "こんにちは")
- список, що містить у собі довільні дані ( ["A", "b", "_"] )
- більш складний об'єкт - про такі трішки пізніше.

## Вимоги

В різних мовах програмування вимоги до змінних різні.

В більш строгих (статично типізованих) мовах ви<br>**не** маєте змоги приліпити стікер спочатку на ціле число,<br>а потім на дробове, або текст.

!!! note ""
    В цих строгих мовах (не Python) при створенні змінної ви вказуєте її тип і він більше не зміниться.

Але змінна залишається змінною,<br>ви можете змінити її **значення**,<br>тобто те **щось** до чого ця наліпка, бірка чи стікер прив'язані.

| Мова | Тип даних | Значення |
| :---: | :---: | :---: |
| Строга | Вказується раз і назавжди | Змінюється в межах одного типу |
| Динамічна | Береться по факту наявних даних, вираховується | Змінюється як саме значення (_часто_) так і його тип (_рідше_, але допускається) |
