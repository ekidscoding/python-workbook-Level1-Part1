Сайт присвячений основам Мови Програмування Пайтон (Python) www.python.org.

Вчимось програмувати на практиці

Частина 1 (цей сайт)

  • programming - Комп'ютерне програмування
  • algorythm - Алгоритм - основа програми
  • variables - Змінні - наклейки, а НЕ коробки
  • data types - Прості типи даних.
    • int - Цілі числпа та операції над ними
    • str - Текстовий тип, робота з рядками
    • float - Дроби та їх обмеження

Частина 2

  • data types - Прості типи даних

    • bool - Умовний тип даних

  • logical operators - вчимо Двійкову Логіку

    • and - Одночасно
    • or - Хоча б одне з
    • not - Протилежне

  • conditionals - Алгоритми з Розгалуженнями

    • if - Проста умова
    • if - else - План "Б"
    • if - elif - else - МультиВсесвіт)

Частина 3

  • data types (list) - Списки. Операції над списками
  • data types (tuple) - Кортежі. Обмеження та користь
  • data types (set) - Множини. Унікальність забезпечена!

Частина 4 вводить поняття "Функції"

  • abstraction - Абстракції
  • def, pass - Мінімальна функція
  • function signature - Що таке сигнатура (шапка) функції
  • return - Конвеєр даних
  • None - Тупик
